Projects

Poo-P

Created at the William Osler KissMyApp Hackathon, it is an android application that gives a stool/urine analysis to the user (informs user about potential health issues along with solutions and prevention of said issues). The app asks the user a series of questions to answer (colour, shape, symptoms etc.) to determine the potential problems.

PythonGrow

This web application was a finalist at the Koding Global Virtual Hackathon. The aim of this application was to introduce software development to beginners through gamification. The web application developed was based on the classic game of snake. Players would navigate the python to eat letters in the correct order to write out a snippet of code. At the end of the level they would receive an explanation of the code snippet

PebbleChange

A Pebble Watch application developed at UofTHacks 2015 based on the theme of social benifit. Users are given a list of charities (choosing from local, national or international) which they can donate their spare change to. They can access more information about the charity through QR codes displayed on the Pebble. The application also showed trending hashtags related to charities on Twitter (such as the ice bucket challenge)

PillowCase

Finalist at WearHacks Toronto 2015. This device integrates the Internet of Things (IOT) with a pillowcase using the Intel Edison/Arduino. The case is used to monitor head movement and snoring. Head movement and snoring are visualized via charts on a website so users and others can identify potential health risks.
